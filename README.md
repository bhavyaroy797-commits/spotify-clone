# spotify-clone

A fully responsive web application that replicates the core UI/UX of the Spotify desktop and mobile experience. Built using core web technologies to practice responsive design principles and CSS layout techniques.

* **Key Features**

-Responsive Layout: Fluid design that adapts seamlessly across mobile, tablet, and desktop viewports using CSS Flexbox and Grid.

-Modern UI: Recreated the iconic dark-theme aesthetic, including sidebars, navigation bars, and playback controls.

-Interactive Components: Styled buttons, hover states, and dynamic elements to simulate a real music streaming environment.

* **Technologies Used**

-HTML5

-CSS3 (custom styling, media queries)
